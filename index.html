Rotina Completa de Manutenção Preventiva do Windows

Siga a ordem sequencial de 1 a 7. **Clique no resumo** para expandir a descrição detalhada e use o botão para copiar o comando exato.

*Todos os comandos **DEVEM** ser executados no **Prompt de Comando (CMD)** ou **PowerShell** como Administrador. O **CHKDSK** (1º comando) requer **reinicialização** do PC.
1º

Verificação do Disco

chkdsk C: /f /r
Clique para ver a descrição detalhada
**Ação Principal:** Este comando é vital para a saúde do seu armazenamento. Ele atua em dois níveis: o parâmetro **`/f`** corrige erros lógicos no sistema de arquivos, e o **`/r`** localiza setores físicos defeituosos, marcando-os para que o sistema não os utilize mais. É o primeiro passo para garantir a integridade da base.
2º

Reparo da Imagem Central (DISM)

DISM /Online /Cleanup-Image /RestoreHealth
Clique para ver a descrição detalhada
**Ação Principal:** O **DISM** (Deployment Image Servicing and Management) conserta a imagem central do Windows, conhecida como *Component Store*. Ele garante que os arquivos de origem usados para reparos (como os do SFC) estejam 100% íntegros, baixando versões saudáveis do **Windows Update** se necessário. É o pré-requisito para um SFC bem-sucedido.
3º

Verificação de Arquivos de Sistema (SFC)

sfc /scannow
Clique para ver a descrição detalhada
**Ação Principal:** O **System File Checker (SFC)** escaneia e verifica a integridade de todos os **arquivos protegidos do sistema operacional**. Se encontrar qualquer arquivo corrompido, danificado ou com versão incorreta, ele o substitui imediatamente por uma cópia em cache (validada pelo DISM).
4º

Limpeza de Componentes Obsoletos

DISM /Online /Cleanup-Image /StartComponentCleanup
Clique para ver a descrição detalhada
**Ação Principal:** Remove versões antigas de componentes do sistema operacional que foram substituídas por atualizações, mas ainda estão armazenadas. É uma forma eficaz de **liberar gigabytes de espaço** na pasta WinSxS (Component Store).
5º

Atualização de Softwares e Drivers (Winget)

winget upgrade --all
Clique para ver a descrição detalhada
**Ação Principal:** Usa o Gerenciador de Pacotes do Windows para **buscar e instalar atualizações** para todos os softwares (e alguns drivers) registrados. É a maneira mais rápida de garantir que a maioria dos seus aplicativos não seja fonte de vulnerabilidade ou erros de performance.
6º

Limpeza de Lixo Eletrônico

del /q /f /s %TEMP%\*
Clique para ver a descrição detalhada
**Ação Principal:** Deleta de forma forçada (`/f`), silenciosa (`/q`) e recursiva (`/s`) todos os arquivos da **pasta temporária do usuário**. Limpar o cache ajuda a reduzir o tempo de acesso a arquivos e otimiza o uso do disco.
7º

Diagnóstico de Bateria (Opcional)

powercfg /batteryreport
Clique para ver a descrição detalhada
**Ação Principal:** (Apenas Notebooks) Gera um relatório HTML completo (salvo em `C:\Windows\system32`) sobre a **saúde da bateria**. Ele compara a **Capacidade Projetada** (quando nova) com a **Capacidade da Última Carga Completa** atual, mostrando o nível exato de desgaste.
